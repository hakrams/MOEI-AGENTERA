Sah Labs OneCX Data Deletion Instructions

Last updated: June 8, 2026

Sah Labs OneCX is a hackathon prototype. Test data can be deleted on request or during routine system cleanup.

How to Request Deletion

To request deletion of prototype test data, email akram@sahlabs.me with the subject line "OneCX Data Deletion Request".

What to Include

Please include the phone number, customer identifier, WhatsApp test number, or case reference used during testing so the related prototype records can be located.

What Will Be Deleted

We will delete matching prototype records where available, including test interaction events, customer profile records, message logs, call transcript text, and related prototype service references.

Processing Time

Deletion requests will be reviewed as soon as practical during the hackathon/testing period. Some backup, security, or server logs may take longer to rotate or may be retained briefly for security troubleshooting.

Questions

For any questions about data deletion, contact akram@sahlabs.me.
